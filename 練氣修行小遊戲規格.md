# 《九劫問仙》練氣修行小遊戲規格

## 共通規則

- 2D直式手機與網頁共同支援；手機使用觸控，網頁使用滑鼠或方向鍵。
- 小遊戲難度以容易理解、短時間完成為主，不依賴複雜連招。
- 每局必須有清楚的操作回饋、進度與結算，不能只在最後顯示隨機結果。
- 完成後依操作表現與隨機事件取得修為；操作不佳或未完成主要目標也不得給零修為。
- 初版共用換算：`獲得修為 = 無條件進位（12 + 操作分數 × 0.36）`，操作分數限制於0～100，因此每局取得12～48修為。
- 平時修行小遊戲沒有二元「成功才給／失敗不給」；成功、普通、失誤只影響操作分數與事件文字。
- 此修為公式不套用於境界突破；突破仍使用有效分數、概率池與既有失敗規則。

## 砍竹／伐木：滑切

- 操作參考水果忍者：玩家按住並快速劃過竹身完成斬擊。
- 每刀依滑動速度、角度、命中位置計算切入量。
- 竹身即時顯示切痕、裂口、竹屑、已切百分比與剩餘厚度。
- 切入量達100%後，竹子向受力方向倒下並完成本輪。
- 連續準確切割提高效率；錯誤不立即失敗，只浪費時間並降低評價。
- 素材：竹林背景、前中後景竹子、完整／受損／斷裂竹身、切面、竹屑、落葉、刀光、倒竹動畫。

## 挑水：水位平衡

- 角色挑著兩桶水行走，水面會受到步伐與路面顛簸影響，產生左右慣性。
- 手機按住畫面左／右區域；網頁使用方向鍵或畫面左右按鈕調整角色重心。
- 水面採彈簧式慣性，玩家過度修正會使水從另一側灑出。
- 水接近桶沿時先顯示浪尖與傾斜；超過桶沿才播放左灑／右灑動畫並扣除水量。
- 畫面只常駐剩餘水量與路程；抵達後依剩餘水量、完成時間與最大傾角結算。
- 素材：移動背景、挑水角色姿勢、扁擔彎曲、左右水桶、水面遮罩、水花、滴水、左右潑灑動畫。

## 顧藥園：限時巡田

- 場上設置六塊藥圃，30～40秒內陸續出現成熟、缺水、蟲害、雜草四種事件。
- 玩家選擇採藥、澆水、驅蟲、除草工具，再點擊或滑過對應植株。
- 同一時間可出現兩至三個事件，形成觀察、判斷與處理優先順序的壓力。
- 成熟藥草拖延會降低藥性；缺水與蟲害持續存在會降低整園健康。
- 錯用工具不立即失敗，只消耗時間並扣除正確率。
- 操作分數依採收量、工具正確率、最終藥園健康與平均反應時間計算。
- 素材：藥園背景、六塊藥圃、成熟／缺水／蟲害／雜草狀態、四種工具、採集、澆水、驅蟲、除草、藥性下降與選錯回饋。

## 參悟心法：記憶小周天

1. 畫面顯示打坐角色剪影、丹田與經脈節點。
2. 青玉光點示範一次小周天運行路線。
3. 示範結束後路線隱藏，只保留角色輪廓與淡色節點。
4. 玩家從丹田按住，以手指或滑鼠連續描出記憶中的路線。
5. 正確經過節點會點亮；順序錯誤或偏離過遠會產生靈氣逸散，但不立即失敗。
6. 路線回到丹田形成閉環即完成一個周天。

- 第一輪4個節點、第二輪5個節點、第三輪6個節點。
- 分數：節點順序45%、描線準確度35%、完成速度20%。
- 素材：打坐角色剪影、經脈線、丹田、節點、示範光點、玩家描線、正確點亮、逸散、周天閉環光效。

## 驗收

- 所有小遊戲必須以實際手指與滑鼠各測試一次。
- 360px寬手機畫面仍能看清主要目標、進度與操作回饋。
- 不閱讀長篇說明也能在一次示範後理解操作。
- 素材、判定與結果必須實際接入遊戲，不接受只有示意圖或文件。
