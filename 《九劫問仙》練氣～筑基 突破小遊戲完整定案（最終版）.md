# 《九劫問仙》練氣～筑基突破小遊戲定案

## 1. 入門至練氣一層

摸問仙石會產生玩家初始四維，但玩家此時仍未正式進入練氣一層。玩家先透過砍竹、挑水、顧藥園、參悟心法等短篇小遊戲累積修為與觸發隨機事件。

- 砍竹：在移動判定線進入斬擊區時點擊。
- 挑水：左右控制平衡，避免水桶中的水灑出。
- 顧藥園：限時辨認成熟藥草、幼苗與雜草。
- 參悟心法：依序點擊短暫出現的經脈符文。
- 每局約20～40秒，完成約2～4局即可達到入門修為，總時間目標2～3分鐘。
- 隨機事件可提供少量額外修為或本次活動修正，但不永久改變境界天花板。

修為達標後觸發「引氣入體」。練氣一層不需要突破、沒有失敗跌層：玩家完成三段引氣操作後即頓悟練氣一層；操作失誤只需重做當前引導，不扣境界。進入練氣一層時，四項屬性各自獨立roll練氣一層成長。

## 2. 通用小境界突破

練氣與筑基每次升層都使用靈氣穩定小遊戲：玩家在浮動指標進入穩定區時點擊或按住，使靈氣維持穩定。

- 練氣1～3層：速度慢、判定區寬。
- 練氣4～6層：速度提高、判定區縮小。
- 練氣7～9層：速度明顯提高、失誤容忍降低。
- 筑基1～3層：高於練氣後段，平滑過渡。
- 筑基4～6層：判定更嚴格。
- 筑基7～9層：高壓操作，為九重雷劫預習。

小境界突破不受靈根或四維屬性影響。

### 成敗與跌層

- 操作分數滿分100。
- 有效分數低於50：突破失敗。
- 練氣二層開始才存在突破失敗；失敗隨機下降1～3層，但練氣境最低保留練氣一層。
- 玩家保留全部屬性，僅修為層數下降。
- 再次突破時依目標層上限判斷；已達上限的屬性增加0，未達上限的屬性取得新成長。

## 3. 問仙石驗證與練氣九層主動墮境

練氣九層圓滿後，玩家可以直接嘗試筑基，或主動下降1～3層重修。主動墮境與突破失敗使用相同規則：層數下降、屬性完全保留。

升層時四維屬性分別與目標層上限比較。顯示「本次屬性提升0」只表示該項能力已達目標層上限，系統不能再給值，不再宣稱能由此證明歷史上每層均為最大roll。

問仙石平時可主動前往查看，也會在玩家準備跨越大境界時自動驗證。完成度採四維總和計算並無條件進位，只作修煉參考提示，不生成永久品階、不改變屬性或突破概率：

```text
目前四維總和 = 靈氣 + 根骨 + 肉身 + 神識
境界四維滿值 = 該境界四項天花板總和
完成度 = ceil(目前四維總和 ÷ 境界四維滿值 × 100)
```

|完成度|評級|
|---|---|
|低於75%|道基不穩|
|75%～84%|小成|
|85%～99%|大成|
|100%|圓滿|

單項高屬性可以補償另一項短板；評語只代表總體完成度。跨境前若結果為「道基不穩」，系統必須顯示二次確認，玩家仍可選擇快速路線直接突破。

## 4. 練氣九層突破筑基

玩家可以裸衝，也可以服用筑基丹。兩種模式使用相同小遊戲，但使用不同概率池。

### 評分

```text
裸衝有效分數 = 原始操作分數
服丹有效分數 = min(100, 原始操作分數 + 15)
```

- 成功／失敗一律以加成後的有效分數判定；有效分數低於50時突破失敗，不進入品階概率池。
- 失敗後在練氣境下降1～3層，屬性全部保留。
- 筑基丹在開始挑戰時消耗，突破失敗也不返還。
- 有效分數50以上：突破成功並進入對應概率池。
- 概率池不生成道基品階名稱，而是決定筑基一層四項獨立roll所使用的成長品質。

### 服用筑基丹概率池

|有效分數|低成長|標準成長|高成長|
|---|---:|---:|---:|
|50～74|70%|30%|0%|
|75～84|15%|85%|0%|
|85～100|0%|60%|40%|

### 裸衝概率池

|有效分數|低成長|標準成長|高成長|
|---|---:|---:|---:|
|50～74|90%|10%|0%|
|75～84|20%|80%|0%|
|85～100|0%|75%|25%|

四項屬性分別獨立抽取品質：低成長取該層roll區間下段60%～70%，標準成長使用完整roll區間，高成長直接取得該層roll最大值。所有小數無條件進位。概率結果沒有名稱，也不改變問仙石評語規則。

「天道極品筑基」已移除，不再提前取得筑基1～3層天花板。

## 5. 筑基一層屬性

跨境成功時，練氣四維屬性定型並完整成為筑基起始基底。練氣組件從此不可再增加，之後只能累加筑基自己的屬性組件：

```text
筑基一層顯示屬性 = 定型練氣組件 + 筑基一層有效成長
```

所有結果受筑基一層「筑基組件上限」裁切。突破概率池只決定本次四項獨立roll的品質，不生成品階，不得直接發放未來層級的屬性，也不得補回未圓滿的練氣組件。

## 6. 程式鎖定規則

- 練氣、筑基小境界突破不吃靈根、不吃四維屬性。
- 靈根只影響突破成功後取得的屬性成長量。
- 四維分別比較目標層上限，不得以四維總和替代。
- 重修只補目前大境界的屬性組件，不能補前一境界的定型缺口。
- 跨大境界後屬性定型，不能跌回上一個大境界。
- 筑基及以上不提供主動墮境，只能因小境界突破或渡劫失敗跌層。
- 所有小數無條件進位。
