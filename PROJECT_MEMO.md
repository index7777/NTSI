# 《九劫問仙》後續設計 Memo

## 修行天數與宗門資源

狀態：僅記錄概念，暫不納入目前 Demo 核心規格，後續另行深入討論。

- 遊戲未來加入「修行天數」概念，用來記錄玩家入宗後經歷的時間與修煉週期。
- 宗門會依固定週期向弟子發放靈石，具體週期、身份差異與發放數量待定。
- 靈石未來可兌換宗門貢獻點。
- 靈石也可直接或間接兌換心法。
- 貢獻點的其他用途、心法品質、修煉消耗及是否影響四維，全部留待後續討論。

待討論問題：

1. 修行天數採現實時間、遊戲行動回合，或兩者混合。
2. 雜役弟子與外門弟子的靈石發放週期及數量是否不同。
3. 靈石兌換貢獻點的比例，以及是否允許反向兌換。
4. 心法是永久被動、修煉效率加成、操作技能，還是新的成長分支。
5. 是否需要宗門俸祿、差事獎勵及月度結算介面。
