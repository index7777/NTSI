# 九劫問仙 2D 原型

目前程式保留一版可玩原型，但橫式正式流程仍在逐階段重製；實際完成與驗收狀態以 `../docs/production/WORK_QUEUE.md` 為準。

## 執行

```powershell
npm install
npm run dev
```

瀏覽器開啟終端顯示的本機網址。手機可在同一區域網路下使用 `npm run dev -- --host 0.0.0.0`，再以電腦區網IP連線測試。

## 驗證

```powershell
npm run validate:balance
npm run build
```

## 已實作

- 原生1200×675、16:9橫式Phaser畫布；舊9:16內容只作過時參考。
- 手機觸控與桌面滑鼠共用操作。
- 男女角色選擇。
- 角色姓名輸入。
- 宗門收徒與問仙石初始四維／靈根生成。
- 砍竹、挑水、顧藥園、參悟心法四種獨立低難度小遊戲。
- 約2～4次活動累積入門修為。
- 引氣入體三段點擊、練氣一層四維獨立roll。
- 雜役弟子身份及慕清璃首次吐納引導。
- 版本化瀏覽器本機存檔與「繼續遊戲」。
- 「道心崩了」功能已取消，程式不提供此入口或刪檔流程。
- 練氣、筑基、金丹數值配置與自動驗算。
- 練氣1～9層修為需求、升層操作評分、四維獨立roll及層級上限裁切。
- 突破失敗下降1～3層、屬性保留與練氣九層主動墮境。
- 問仙石四維總和評級及道基不穩提示。
- 柳如煙首次事件、筑基丹×2、裸衝／服丹雙概率池。
- 筑基失敗掉回練氣6～8層；成功晉升筑基一層與外門弟子。
- 舊宗門雜役庭院與筑基洞府9:16素材只作內容參考，正式場景需重製為橫式分層版本。
