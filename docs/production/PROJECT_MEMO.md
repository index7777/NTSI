# 《九劫問仙》後續設計 Memo

## 修行天數與宗門資源

狀態：僅記錄概念，暫不納入目前 Demo 核心規格，後續另行深入討論。

- 遊戲未來加入「修行天數」概念，用來記錄玩家入宗後經歷的時間與修煉週期。
- 宗門會依固定週期向弟子發放靈石，具體週期、身份差異與發放數量待定。
- 靈石未來可兌換宗門貢獻點。
- 靈石也可直接或間接兌換心法。
- 貢獻點的其他用途、心法品質、修煉消耗及是否影響四維，全部留待後續討論。

待討論問題：

1. 修行天數採現實時間、遊戲行動回合，或兩者混合。
2. 雜役弟子與外門弟子的靈石發放週期及數量是否不同。
3. 靈石兌換貢獻點的比例，以及是否允許反向兌換。
4. 心法是永久被動、修煉效率加成、操作技能，還是新的成長分支。
5. 是否需要宗門俸祿、差事獎勵及月度結算介面。

## NPC待辦

- 男性NPC人設與角色母板待後續補充。

## 進入玩法前的劇情場景卡

狀態：新手服裝版本先行製作；跨境版本列為待開發。

- 場景卡不是實際玩法截圖，而是玩家進入活動前的敘事插畫。
- 每張卡必須依玩家性別顯示對應角色，並遵守已驗收的角色母板。
- 待練氣、築基及後續跨境服裝確認後，依「性別 × 境界 × 活動」建立不同版本。
- 境界提高時，卡片同步調整服裝、靈氣特效、動作氣勢及場景演出，不可沿用錯誤服裝。
- 同一活動在不同境界的卡片演出可以升級，但小遊戲操作方式暫時維持一致，除非另有玩法改版規格。
- 新版本必須依 `docs/art/CHARACTER_PORTRAIT_STANDARD.md`、角色母板及根目錄 `AGENTS.md` 驗收。

## 橫式場景三時段待開發

- 首圖、宗門庭院、問仙石、洞天／閉關及築基洞府需製作卯時、午時、酉時三個同鏡頭橫式版本。
- 小遊戲暫時只製作基準版本；玩法與美術定案後，才開始午時與酉時版本。

## 小遊戲統一時限

- 基礎時限 30 秒並顯示倒數。
- 時間到未完成成功條件，包括未操作及只完成部分進度，均判定失敗。
- 失敗仍依有效操作結算少量修為；是否改為零修為需另立數值規格。

## 母板示意但不開發的功能

- 「宗門事務」與「散步遊歷」只存在於介面母板示意，不屬於目前或待開發的實機功能。
