## 選角畫面不可改動的規則

- 人物與場景使用已烘好的人景合成背景；不要再疊獨立男女角色，避免融圖不自然。
- 其他場景圖都留在預備／候選區，不得刪除。
- 頂部中央程式文字「選擇角色」；其下細線與中央菱形為獨立裝飾素材。
- 左上「返回」、右上「設定」文字由程式渲染。
- 跨場景控制項拆成獨立透明素材：金圈、返回箭頭、齒輪。圈內圈外都透明，不得烘入黑底。
- 圖示下方看似水墨的黑色橫向方塊不是圖片素材；由 CSS／Phaser 畫半透明矩形。
- 左側資訊欄只用透明金色線框；黑色半透明方形底由 CSS／Phaser 畫，文字由程式渲染。
- 左右選擇條文字固定為 `男　◇ 選擇此角色`、`女　◇ 選擇此角色`。
- 點男女選擇條只切換選擇狀態；底部「確認選擇」才進入取名。不要點性別後立刻跳頁。
- 確認選擇按鈕使用獨立圖底，文字由程式渲染。
- 版面須盡量與母板一致，不能用一般 CSS 膠囊、臨時紙卷或不相符按鈕代替。

## 取名畫面不可改動的規則

- 沿用同一張烘好人物與場景背景，以及拆層的左上返回、右上設定。
- 中央使用名冊／宣紙面板構圖。
- 移除輸入框上方「姓名」及左右分隔線；不做輸入框右側重置按鈕。
