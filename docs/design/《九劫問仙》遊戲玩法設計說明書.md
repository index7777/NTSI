# 《九劫問仙》遊戲玩法設計說明書

## 1. 專案定位

- 中文名：九劫問仙
- 英文名：Nine Tribulations Seek Immortality
- 簡稱：NTSI
- 類型：修仙養成、境界突破、天劫操作闖關、隨機NPC劇情
- 平台：手機瀏覽器與桌面網頁瀏覽器
- 畫面形式：2D直式遊戲，主要基準比例9:16，桌面網頁置中顯示並補背景留白
- 輸入：所有必要操作均支援單點觸控與滑鼠；鍵盤只作桌面額外快捷鍵，不得成為通關必要條件
- 核心口號：練氣凡軀無劫，筑基圓滿問天
- Demo終點：金丹九層圓滿後嘗試突破，播放 Demo Ending；元嬰內容不開放。

## 2. Demo流程

```text
摸石測靈根
→ 砍竹／挑水／顧藥園／參悟心法
→ 引氣入體、頓悟練氣一層
→ 練氣1～9層修煉與突破
→ 主動重修或嘗試筑基
→ 柳如煙贈丹事件
→ 裸衝／服丹突破筑基
→ 筑基1～9層修煉
→ 九重雷劫
→ 金丹1～9層修煉
→ 金丹圓滿突破演出
→ Demo Ending
```

開局建立角色時，玩家可選擇男性或女性主角。性別只改變角色立繪、動畫、稱謂及部分NPC對話代稱，不影響初始四維、靈根概率、成長倍率、操作難度或任何境界天花板。

## 2.1 角色美術母板

- 統一風格：2D水墨仙俠背景、柔和水彩上色、清楚墨線、約3.5頭身Q版比例、表情易讀，適合手機小畫面。
- 男主角：黑髮實用高髻、灰白與青灰雜役修煉服、護腕與布靴，樸素、親切、帶初入宗門的朝氣。
- 女主角：與男主角同門同階、同材質與同功能服裝，採實用半束髮；不可用華服或性感化設計造成身份差異。
- 柳如煙：淺煙紫收腰宗門道袍、銀灰腰帶、柳葉暗紋、白玉柳釵、青瓷雕花丹瓶；外表刻意溫柔精緻，眼尾微挑並帶評估意味。
- 柳如煙表情組：偽裝甜笑、暗中盤算、玩家失敗時不耐嫌棄、玩家成功時討好巴結。
- 角色辨識：玩家角色以灰白青灰為主，柳如煙是主要角色中唯一大面積使用煙紫、柔絲長袖與精緻收腰剪裁者。

## 3. 境界規則

- 練氣全程無天劫。
- 練氣一層由入門修為達標後直接頓悟，不需要突破，也不存在失敗跌層。
- 練氣九層突破筑基可裸衝，也可使用筑基丹。
- 筑基一至八層無天劫。
- 筑基九層圓滿觸發九重雷劫，成功進入金丹。
- 跨大境界後，上一境界屬性定型並成為新境界基底；未達天花板的缺口永久保留，不能由下一境界補足。
- 跨境後不會跌回上一個大境界。
- 當前境界突破失敗只下降該境界1～3層，保留全部屬性。

### 3.1 身份稱號

- 尚未練氣及練氣1～9層：雜役弟子。
- 練氣九層突破筑基成功：晉升外門弟子。
- 筑基與金丹階段暫時都沿用外門弟子稱號。
- Demo不開放內門弟子，也不因境界自動產生內門身份。
- 身份稱號主要用於服裝、介面、NPC稱呼與演出，不額外增加數值。

## 4. 四維屬性

|屬性|主要功能|
|---|---|
|靈氣|護盾容量與修煉效率|
|根骨|最大生命值換算|
|肉身|雷擊命中後的傷害減免|
|神識|提前預測後續QTE與雷擊位置|

四維皆可參與最大生命值的基礎換算，但根骨權重最高。建議初版公式：

```text
最大生命值 = ceil(
  100
  + 靈氣 × 0.10
  + 根骨 × 0.35
  + 肉身 × 0.25
  + 神識 × 0.05
)
```

係數在試玩後調整。

## 5. 靈根

|靈根|屬性成長倍率|
|---|---:|
|雜靈根|0.6|
|三靈根|0.8|
|雙靈根|1.0|
|單靈根|1.3|

所有乘法小數無條件進位。靈根影響修煉效率及每次取得的屬性成長，不影響突破小遊戲難度，也不改變境界硬天花板。低靈根可透過跌層重修持續補足屬性。

## 6. 修煉、突破與重修

- 入門階段透過砍竹、挑水、顧藥園、參悟心法累積修為，約2～3分鐘後引氣入體並頓悟練氣一層。
- 練氣二層開始，每次升層前進行小突破遊戲。
- 有效分數低於50視為失敗，隨機下降1～3層。
- 屬性不因跌層減少。
- 再次升層時，四維分別與目標層上限比較。
- 已達目標層上限的屬性不再成長。
- 未達目標層上限的屬性取得新成長並累加，結果裁切至目標層上限。
- 練氣九層允許主動墮境1～3層；筑基與金丹不提供主動墮境。

完整公式與天花板以《數值累加邏輯修復終版》為準。

## 7. 筑基突破

- 玩家可裸衝或服用筑基丹。
- 筑基丹使操作有效分數增加15，最高100，並改用服丹概率池。
- 成敗一律看加成後的有效分數；有效分數低於50時突破失敗，下降練氣1～3層並保留屬性。
- 有效分數50以上：突破成功，再依分數及模式為筑基一層四項屬性分別抽取低、標準或高成長。
- 概率池不產生品階名稱。問仙石的道基不穩、小成、大成、圓滿只依目前四維總和判定，僅作提示。
- 天道極品筑基已移除。
- 練氣屬性在跨境時定型，成為筑基基底；再疊加筑基一層成長。筑基只能增長筑基組件，不能補足練氣缺口。

詳細概率以《練氣～筑基突破小遊戲定案》為準。

## 8. NPC劇情

首位NPC為柳如煙。她在玩家練氣九層圓滿時登場，以贈送筑基丹投資玩家，並依玩家後續成功、失敗或渡劫狀況改變態度。

柳如煙只提供突破資源與劇情人情，不直接保證突破或渡劫成功。後續可擴充耿直、冷漠、高傲等不同立場NPC。

第二位定型NPC為慕清璃：外門第一天才，冰冷寡言，對所有弟子一視同仁地嚴格，不假辭色。她負責以極短台詞引導玩家第一次吐納，也可在後續修煉事件中提供精準但不帶安慰的評語。慕清璃使用固定服裝，不隨玩家境界換裝。

男性NPC暫列企劃Memo，待後續補充人設後再建立正式角色。

## 9. 問仙石

問仙石第一次使用時產生初始四維；之後玩家可隨時主動前往驗證目前修煉完成度。每次準備跨越大境界時系統也會自動驗證。

世界觀定位：宗門收徒必須先以問仙石辨認來者是否具有靈根，並留下初入仙途時的根基記錄。因此第一次摸石是入門資格驗證；後續摸石是以同一份記錄衡量玩家目前的修煉完成度，不會重抽或改變靈根。

```text
完成度 = ceil(目前四維總和 ÷ 當前境界四維滿值總和 × 100)
```

- 低於75%：道基不穩。
- 75%～84%：小成。
- 85%～99%：大成。
- 100%：圓滿。

評語允許單項高值補償另一項短板，只是修煉參考，不影響突破概率。跨境前若為道基不穩，系統顯示確認視窗，但允許玩家繼續突破。

## 10. 九重雷劫

### 10.1 通關與失敗

- 閃避成功完全免傷。
- 玩家生命值歸零則渡劫失敗。
- 雷劫之間不自動回血。
- 不提供根骨被動回血。
- 玩家可使用回血丹藥；Demo初版攜帶與使用次數無上限、無冷卻，使用時遊戲不暫停。
- 第九劫完成小遊戲即成功；若未完成閃避但承受雷擊後仍存活，也算成功。
- 渡劫失敗後，以筑基九層為基準下降1～3層，因此落到筑基八、七或六層，屬性全部保留。

### 10.2 九劫雷擊數量

|劫數|雷擊數量|定位|
|---|---:|---|
|第一劫|1道|教學與節奏認識|
|第二劫|3道|基礎連續閃避|
|第三劫|6道|初劫收尾|
|第四劫|8道|進入中段壓力|
|第五劫|9道|穩定連擊|
|第六劫|9道|縮短預警時間|
|第七劫|9道|高速終劫|
|第八劫|9道|複合落雷位置|
|第九劫|全螢幕最大雷擊|最終承傷／操作檢定|

雷擊基礎傷害區間：第一至三劫40～60、第四至六劫70～110、第七至九劫130～180。

- 第五劫：每3道主雷追加1道50%傷害小雷。
- 第六劫：每3道主雷追加2道50%傷害小雷。
- 第七劫：9道主雷外加入3次75%傷害追蹤範圍雷。
- 第八劫：9道主雷外加入4次75%傷害追蹤範圍雷及小雷干擾。
- 玩家可用三區閃避、精準彈反或護盾硬扛處理攻擊。

### 10.3 傷害與護盾

神識不減傷，也不提供固定傷害豁免。

```text
肉身減傷率 = min(0.50, 肉身 ÷ 筑基肉身天花板 × 0.50)
命中傷害 = ceil(基礎雷傷 × (1 - 肉身減傷率))

護盾容量 = ceil(靈氣 × 護盾換算係數)
護盾扣除 = min(目前護盾, 命中傷害)
生命扣除 = max(0, 命中傷害 - 目前護盾)
```

- 沒有受傷時不消耗護盾，護盾可維持到下一劫。
- 護盾不在劫與劫之間自動恢復。
- 初版護盾換算係數建議為0.5，試玩後調整。
- 護盾整場只在開始時生成一次，劫與劫之間不恢復。
- 護盾完全吸收命中傷害時不產生硬直。
- 護盾被打破且傷害穿透生命時，產生3秒破盾硬直；硬直期間不再次疊加硬直，破盾後另有0.75秒受擊保護。

### 10.4 QTE與神識預測

QTE（Quick Time Event）是限時操作事件：畫面在特定位置顯示點擊、滑動或按住指令，玩家必須在時間窗結束前完成。成功則閃避該次雷擊，失敗則受到雷擊。

神識不直接增加QTE判定寬度，而是提前揭示未來事件：

|神識完成度|可預測內容|
|---|---|
|低於25%|只顯示當前雷擊位置|
|25%～49%|提前顯示下一個雷擊／QTE位置|
|50%～74%|提前顯示後續2個位置|
|75%～99%|提前顯示後續3個位置及類型|
|100%|提前顯示後續4個位置、類型與順序|

神識完成度計算結果採無條件進位。

彈反是在雷擊命中前點擊其預告區域，成功時完全免傷。初版窗口：初劫0.35秒、中劫0.25秒、終劫0.18秒。神識只提供位置與順序預測，不延長彈反窗口。

### 10.5 受擊硬直

初版建議值：

- 第一至三劫：0.35秒。
- 第四至六劫：0.45秒。
- 第七至九劫：0.60秒。
- 同一次連擊期間硬直不重複疊加，只刷新到該階段最大值。
- 硬直期間仍顯示神識預測，但不能執行閃避輸入。

硬直時間屬試玩參數，若產生連續命中無法脫離的情況，優先縮短硬直或加入受擊後短暫無敵幀。

### 10.6 2D跨平台操作

- 戰鬥區分為左、中、右三個觸控區，點擊安全區進行閃避。
- 雷擊預告、QTE位置與神識預測都使用2D地面標記，不依賴精細文字閱讀。
- 彈反與護盾使用畫面下方固定大按鈕，觸控熱區不得小於視覺按鈕。
- 手機直式畫面優先；桌面網頁沿用相同比例與UI位置，可用滑鼠點擊。
- 桌面可選配方向鍵／A、S、D閃避及空白鍵彈反，但所有內容必須能只用滑鼠完成。
- 暫停、丹藥及系統選單避開手機安全區與瀏覽器底部工具列。

## 11. 道具

Demo保留兩類核心丹藥：

- 筑基丹：筑基突破有效分數+15並切換概率池，開始挑戰時消耗。
- 回復丹藥：九重雷劫中依道具設定恢復最大生命百分比，不可恢復護盾；攜帶與使用次數無上限、無冷卻，使用時遊戲不暫停。Demo若因此過於簡單再調整。

不製作煉丹、材料採集或裝備系統。

## 12. 後續劫難定位

九重雷劫只是筑基升金丹的第一種大境界考驗，不是整個遊戲唯一核心。後續境界可使用心魔、幻境、因果、戰鬥或其他劫難，不固定重複雷劫。

金丹九層圓滿後播放未知新劫難的天地異象與 Coming Soon 動畫，不進行另一場雷劫，也不判定成功或失敗，隨後進入 Demo Ending。

## 13. Demo內容邊界

包含：練氣、筑基、九重雷劫、金丹1～9層、柳如煙事件、基礎存檔與Demo Ending。

不包含：元嬰內容、一般戰鬥、裝備、煉丹製作、材料副本、多人系統。

後續規劃但暫不實作：修行天數、宗門定期發放靈石、靈石兌換貢獻點、靈石或貢獻點兌換心法。細節統一記錄於 `../production/PROJECT_MEMO.md`，待後續討論。

存檔功能：網頁與手機版使用版本化本機存檔，標題畫面提供「繼續遊戲」。主介面的「道心崩了」會詢問「確定放棄這一世，重新來過？」；確認後刪除全部本機進度並重新開局，不可復原。
