# 清心訣：經脈共鳴小遊戲規格

## 共用 HUD

- 清心訣使用共用 `StreakCounter`，傳入 `label="共鳴"`。
- 輸入介面：`label`、`value`、`tier`、`visible`。
- 元件不得寫死遊戲名稱，只負責共用位置、淡金色數字、進場回彈、歸零淡出、暫停與 Responsive 表現。
- `tier` 門檻暫定：0～9 一般、10～19 中階、20 以上高階。

## 經脈座標校準

邏輯畫布固定為 1200 × 675；畫面縮放不得改變判定。

| 順序 | 穴位 | X | Y |
|---:|---|---:|---:|
| 1 | 百會 | 476 | 259 |
| 2 | 神庭 | 474 | 312 |
| 3 | 膻中 | 471 | 376 |
| 4 | 關元 | 466 | 446 |
| 5 | 氣海 | 464 | 530 |

穴位不是完全垂直排列，必須依古籍內實際經脈線的微斜角移動。

## 游走方向

- 初始位置：百會。
- 第一個判定目標：神庭。
- 路徑：百會 → 神庭 → 膻中 → 關元 → 氣海 → 關元 → 膻中 → 神庭 → 百會。
- 到達上下端點時反向，不跳躍、不瞬間回到另一端。
- Miss 只將共鳴歸零，不中斷經脈往返。

## 共鳴加速係數 v1

| 目前連續共鳴 | 每穴基礎間隔 |
|---:|---:|
| 0～4 | 1300 ms |
| 5～9 | 1200 ms |
| 10～19 | 1100 ms |
| 20～29 | 1000 ms |
| 30 以上 | 920 ms |

- 最低間隔硬限制：900 ms。
- 每次加入 ±50 ms 小幅變化，避免玩家只按固定拍子。
- 判定時間差：Perfect 70 ms、Good 150 ms、Normal 250 ms、Miss 大於 250 ms。
