# 《九劫問仙》雷劫QTE鏡頭與速度感規格

> 狀態：過時原型／暫緩。本文不屬目前正式玩家流程；里程碑止於筑基九層圓滿。未經使用者解除暫緩，不得依本文製作或接入。舊直式敘述亦無效，未來重啟時須改為原生1200×675橫式重新驗收。

## 1. 鏡頭分層

- 雷劫世界場景與HUD使用獨立鏡頭／顯示層。
- 世界鏡頭可縮放、偏移與震動；生命、護盾、劫數、QTE提示和道具按鈕固定，不跟著縮放。
- 鏡頭效果不得改變實際碰撞區與觸控按鈕位置。

## 2. QTE鏡頭節奏

### 預警

- 下一次主要雷擊或QTE即將發生時，世界鏡頭在180～260ms內輕微Zoom In。
- 聚焦點位於角色與預測雷擊位置之間；神識顯示的預測位置需保持可見。
- 預警Zoom不得遮住逃生方向。

### 閃避

- 成功輸入後鏡頭沿角色移動方向偏移，並在220～360ms內Zoom Out。
- 角色顯示短殘影、地面速度線與雲霧反向視差。
- 閃避失敗時不播放成功拉鏡，改為短促受擊壓縮與震動。

### 彈反

- 成功接觸雷擊瞬間停格40～70ms。
- 停格後快速Zoom Out並讓雷光向外擴散，表現反作用力。
- 彈反失敗則播放受擊，不得沿用成功衝擊演出。

### 護盾

- 護盾承受雷擊時維持較近鏡頭，使用護盾曲面變形、亮度壓縮與低幅震動表現重量。
- 護盾破裂時加入一次較強震動與碎片外推，角色進入3秒硬直。

## 3. 速度感來源

- 鏡頭Zoom In／Zoom Out。
- 世界鏡頭方向偏移。
- 前中後雲層不同速度視差。
- 角色殘影、衣髮延遲與地面速度線。
- 雷擊前短暫收束、命中瞬間停格、結果快速釋放。
- 不使用長時間全螢幕動態模糊，避免手機效能與暈動問題。

## 4. 強度限制

- 一般QTE鏡頭縮放建議限制於1.00～1.12。
- 大型雷擊最高可短暫到1.16，但持續時間不得超過500ms。
- 非受擊狀態不使用高頻連續震動。
- 第五至第八劫的多道雷擊需安排主次節奏，不能每一道雷都使用完整Zoom演出。
- 第九劫全螢幕最大雷擊可使用一次完整鏡頭收束與釋放。

## 5. 降低鏡頭動態

- 設定中提供「降低鏡頭動態」。
- 開啟後縮放幅度降低至少60%，移除快速方向偏移與高頻震動。
- QTE預警時間、碰撞判定與操作分數不得因設定而改變。

## 6. 驗收

- 過時驗收條件：原「手機直式」要求已取消。未來重啟時改測1200×675、960×540、640×360橫式。
- Zoom過程不得露出場景貼圖邊界。
- 閃避、彈反、護盾與受擊必須能只看演出就分辨結果。
- 鏡頭演出不影響QTE有效分數計算。
