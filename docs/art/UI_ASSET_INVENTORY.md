# UI／UX／HUD 資產拆分清單

狀態標記：`planned` 待製作、`candidate` 候選、`approved` 已驗收。

| 系統 | 獨立資產 | 狀態 | 程式功能 |
|---|---|---|---|
| 頂部 HUD | 宣紙底板、玩家頭像框、時段節點、修為條外框 | planned | 日期、時段、境界與修為即時更新 |
| 功能入口 | 人物、背包、日誌、設定圖示 | planned | 開啟對應功能頁 |
| 修行入口 | 砍竹、挑水、藥園、心法圖示 | planned | 解鎖、選取、紅點與進入流程 |
| 場景卡 | 卡片紙框、返回圖示、開始修行按鈕狀態 | planned | 顯示性別／境界對應插畫及活動資訊 |
| 小遊戲 | 操作 HUD、暫停、倒數、分數、回饋元件 | planned | 實際操作與判定 |
| 對話 | 對話框、名稱牌、選項按鈕狀態 | planned | 台詞、選項與劇情流程 |
| 日誌 | 紙卷面板、日期分隔、捲動提示 | planned | 活動紀錄與統計 |
| 結算 | 紙卷面板、分隔紋樣、繼續按鈕 | planned | 修為、次數、時段與跨日結算 |
| 系統選單 | 設定面板、音量控制、確認視窗 | planned | ESC、設定、存檔、自毀道心與離開 |
| 場景移動 | 右下入口圖示、正常／懸停／按下／鎖定狀態 | planned | 未開放時提示「宗門權限不足，無法前往」 |

任何資產改為 approved 前，必須完成原尺寸、960×540、640×360、滑鼠、鍵盤與觸控驗收。
