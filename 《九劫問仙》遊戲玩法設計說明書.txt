《九劫問仙》遊戲玩法設計說明書
1. 專案基礎資訊
遊戲中文名：九劫問仙
遊戲英文名：Nine Tribulations Seek Immortality
專案簡寫：NTSI
遊戲類型：修仙養成 + 境界突破 + 天劫操作闖關 + NPC隨機劇情爽文小游戏
核心定位：還原凡人修仙境界規則、摒除不合理練氣渡劫設定，結合「隨機心機師姐人情博弈」+「後期九重雷劫核心操作」，輕量級、節奏舒服、爽點明確、無版權風險
核心口號：練氣凡軀無劫，筑基圓滿問天
2. 世界觀與核心規則（硬性定死、全程不衝突）
2.1 境界天道規則（完全還原凡人修仙正統設定）
整套規則1:1搬用凡人修仙核心天道邏輯，捨去魔改設定，所有劫數、境界桎梏完全對標原著，同時适配本遊戲玩法機制，規則絕對自洽：
1. 凡境（練氣、筑基）皆為凡仙根基：未踏足金丹之前，修士依舊屬於「凡修」，天地天道干涉極弱，無主動降下劫罰。
2. 練氣1–9層：徹底無劫、無天道考驗：僅為積累靈氣、洗鍊凡軀的基礎過程，無任何天道風險，修為滿層即可獲得突破資格。
3. 練氣突破筑基：無天劫、無丹毒，純運氣突破：筑基丹為凡境最高階突破丹藥，僅用於打破凡胎道基桎梏；突破成敗完全隨機，只看修士自身仙緣運氣，失敗耗丹掉修為，無任何天道懲罰與丹毒反噬。
4. 筑基1–8層：穩固道基、積累靈氣：屬於境界過渡階段，天道默認凡修成長，不會觸發任何劫雷與考驗，僅可穩定提升修為屬性。
5. 筑基9層圓滿：凡修最後桎梏，強制觸發【九重筑基雷劫】：脫離凡修、邁入真正仙途的第一道天道門檻，是凡境唯一、也是第一道天劫，渡劫成敗決定能否踏入金丹大道。
6. 天道核心邏輯：凡修無劫，成仙始劫；劫數不隨境界晉升而來，只隨「脫凡入聖」的道基蛻變而生。
1. 練氣1–9層：絕無任何天劫、雷劫，僅為積累靈氣、打磨凡軀，天道不會降下任何考驗，徹底還原原著邏輯，不強加劫數。
2. 練氣突破筑基：僅依靠筑基丹重塑道基，為純粹修為突破，無渡劫、無雷戰、無丹毒，成敗只看隨機運氣。
3. 筑基1–9層：正常積累修為，無特殊天道考驗。
4. 筑基9層圓滿：觸發九重雷劫（遊戲唯一核心渡劫玩法），是全遊戲操作難點、爽點、核心關卡。
2.2 突破核心邏輯
丹藥只負責「開啟境界門票」，不代表突破必成功、不代表未來能渡劫。
人情是人情，實力是實力：NPC贈丹是施恩押寶，玩家能否成事全看自身修為與操作。
3. 完整遊戲流程線（標準循環）
打坐累積修為 / 日常修煉 → 練氣9層滿級 → 隨機觸發師姐NPC劇情領取筑基丹 → 洞府閉關突破（有失敗機率）→ 成功晉升筑基1層 → 筑基1–9層養成積累 → 筑基滿級觸發九重雷劫渡劫 → 渡劫成功進入下一大境界 / 渡劫失敗跌落重修
4. 完整境界系統（凡人正統體系 + 遊戲專屬機制補全）
目前遊戲開放階段：練氣境、筑基境，後續可無縫拓展後續境界，所有境界機制、桎梏、規則完全對標凡人原著。
4.1 練氣境（1–9層）【凡軀無劫境】
境界定位：最基礎修仙境界，脫離凡人生老病死的初步修煉階段，全程無天道干涉。
- 獲取修為：自動打坐、手動周天運氣、日常修煉積累
- 階段特性：徹底凡胎之軀，靈氣儲量有限，無任何天劫、災厄、天道考驗
- 滿級機制：練氣9層圓滿後，鎖定修為上限，無法自行突破，強制觸發隨機師姐NPC劇情
- 突破前置：練氣9層滿修為 + 持有筑基丹（唯一突破媒介）
- 突破機制：純隨機運氣判定，無人為操作影響、無屬性加成影響
- 突破懲罰：突破失敗，筑基丹全额消耗、修為小幅倒退，無任何額外懲罰
- 境界特點：門派基礎弟子标配境界，修為浮淺、無道基、無仙途資質認證
4.2 筑基境（1–9層）【道基初成、凡極生劫】
境界定位：脫離凡胎、凝聚道基，正式踏入仙途門檻，是凡修與準仙的分水嶺。
- 階段特性：丹田重塑、道基凝實，靈氣轉為純正仙氣，具備引動天道感知的資格
- 1–4層（筑基初境）：穩定道基、適應仙氣體質，修為提升平穩，無任何風險
- 5–8層（筑基中高境）：積累道基厚度、擴容靈氣儲備，純養成階段，無天道考驗
- 9層圓滿（筑基巔峰）：凡修境界極限，道基完全成型，強制觸發九重筑基雷劫
- 渡劫機制：遊戲核心操作玩法，分段九重雷劫、閃躲+階段QTE判定
- 渡劫成敗獎懲：
成功：道基蛻變完美，解鎖後續金丹境界資格，屬性大幅提升
失敗：道基損毀、境界跌落隨機筑基層數，需重新積累修為修復道基
4.3 後續可拓展完整境界序列（凡人正統體系，預留開發空位）
完整仙途境界鏈：練氣 → 筑基 → 金丹 → 元嬰 → 化神 → 煉虛 → 合體 → 大乘 → 渡劫飛仙
當前Demo開放境界：練氣 → 筑基 → 金丹9層圓滿封頂
5. 角色屬性系統（Demo 練氣～金丹9層 完整可落地設計）
設計核心原則：完全參照凡人修仙體系、輕量簡潔、不卡突破、不做數值膨脹亂數、屬性只影響修煉速度與渡劫容錯，突破成敗只看運氣，與屬性無關。
四大基礎屬性（全階段通用）
1. 靈氣底子：影響打坐修為獲取速度、靈氣上限
2. 根骨資質：修仙天賦，越高每層升級額外屬性加成越高
3. 肉身體質：雷劫減傷專屬屬性，渡劫核心屬性，體質越高雷劫容錯越高
4. 元神神識：影響QTE判定寬容度、反應速度、躲劫精準度
5.1 屬性規則（硬性定死）
- 小境界升層（1–9層）：自動小幅屬性成長
- 大境界突破（練氣→筑基→金丹）：大幅屬性蛻變、數值翻倍提升
- 突破成功率 100% 不受屬性影響，只隨機運氣判定
- 屬性只影響：修煉效率、渡劫難度容錯、QTE手感
5.2 全境界屬性機制對應（凡人設定＋Demo閉環）
5.2.1 練氣境（1–9層）凡軀階段
- 特性：神識弱小、肉身凡胎、靈氣淺薄
- 屬性作用：只影響打坐速度，無任何渡劫效果（練氣無劫）
- 練氣9層滿級：屬性達凡軀頂峰，等待丹藥突破
5.2.2 筑基境（1–9層）道基成型
- 特性：重塑丹田、靈氣轉仙氣、神識開竅、肉身強化
- 屬性新增功能：體質、神識開始影響雷劫容錯
- 筑基1–8層：屬性穩定累積
- 筑基9層巔峰：屬性拉滿，準備九重雷劫（屬性決定你好不好渡劫）
5.2.3 金丹境（1–9層）丹成固道【Demo最終封頂】
- 特性：結丹凝道、靈氣固化、肉身超凡、神識大增、徹底脫離凡修
- 境界規則：金丹前中期無劫
- 金丹9層圓滿：預留未來元嬰天劫觸發口（Demo暫不開放）
5.3 屬性對戰/渡劫機制（核心爽點）
1. 體質 = 雷劫減傷
體質越高，被雷劫命中扣血越少，新手容錯極高
2. 神識 = QTE判定寬度
神識低：判定極嚴、容易失敗；神識高：判定寬鬆、手感順滑
3. 根骨 = 升層加成倍率
同樣滿級，高根骨玩家屬性會比普通玩家高一大截
4. 靈氣 = 修煉速度
縮短掛機打坐時間，不影響突破機率
5.4 隨機初始資質（重開檔差異、提升重玩價值）
開局隨機根骨檔次（對應爽文修仙套路）：
- 凡品根骨：成長低、修煉慢、渡劫難
- 良品根骨：標準體驗、平衡數值
- 上品根骨：高成長、渡劫容錯高
- 絕品靈根：天胡開局、屬性拉滿、渡劫極簡（歐皇專屬）
5.5 補充：境界對應天道權限（再次強化凡人規則）
- 練氣：凡軀，無天道權限、無劫、無仙識
- 筑基：初入仙途，唯巔峰引劫
- 金丹：穩固道果、脫凡入聖，Demo最終閉環境界
6. 道具系統（簡化、無多餘雜項）
僅保留核心道具，無煉丹系統、無禁地刷材料、無繁雜養成
5.1 筑基丹（唯一低階關鍵道具）
- 獲取方式：練氣9層滿級，隨機師姐NPC贈送
- 作用：唯一練氣突破筑基的媒介
- 機制：只開啟突破資格，不保障成功率，不參與後續渡劫
- 消耗規則：突破成功/失敗均會消耗丹藥
6. NPC隨機劇情系統（核心亮點：爽文人情博弈）
機制說明：玩家每次練氣9層滿級，隨機刷新一位門派師姐贈送筑基丹，不同師姐性格、台詞、對玩家態度完全不同，提升重玩價值。
首批定型NPC：柳如煙（心機綠茶投機型）
6.1 柳如煙 完整人設定型
人設標籤：爽文經典惡女、偽善溫柔、功利押寶、抱大腿、雙標現實
核心性格：無真心善意，贈丹是「投資潛力新人」，玩家強則巴結，玩家弱則翻臉冷漠，精於人情算計。
外觀設定（最終定稿）
- 面容：清秀柔媚，眼尾上挑，慣用淺笑偽裝溫和，算計時眼底藏精明，笑裡藏刀
- 髮型：半垂雲髻，不似苦修弟子樸素，髮間點綴白玉柳釵，對應名字氣韻
- 穿搭：淺煙紫軟綢道袍，門派制式但私自剪裁收腰，氣質嫵媚精緻，區別於清苦修煉的師姐；袖口繡細小柳葉紋，常年手持青瓷丹瓶
- 肢體習慣：遞丹時刻意溫柔俯身打量玩家氣息、判斷潛力；玩家失意時下意識後退拉開距離，情緒完全隨玩家實力浮動
首次登場劇情（練氣9層滿級觸發）
柳如煙帶著刻意柔和的笑意上前，手捧青瓷丹瓶遞來：
「師弟好本事，居然修至練氣九層圓滿。這兩顆筑基丹，是我專門為你留的。」
隨後語氣平實講清規則、暗藏人情綁定：
「你須記住，練氣九層依舊是凡胎，本無天劫。唯有藉此丹重塑道基，方能踏入筑基。丹藥只做突破媒介，能否成功突破全看個人運氣，運氣不足則會突破失敗、修為倒退。
至於九重雷劫，那是你日後筑基圓滿才需面對的考驗，眼下無須顧慮。」
最後拋出功利人心伏筆：
「今日我贈你丹藥、成人情一份，將來你若筑基有成、踏過雷劫，可千萬記得提攜於我。」
三段後續專屬互動（強化雙標人設）
1. 衝關失敗重修滿級
柳如煙面露不耐，不情願取出一顆丹藥：「連突破都做不穩，真是不堪大用。再給你一次機會，別辜負我人情。」
2. 渡劫失敗、境界跌落重修
笑意盡散、態度冷淡：「雷劫都扛不住，看來我當初押錯人了。想再求丹，先磨出實力再說。」
3. 筑基九層滿級、即將渡劫
再度偽裝溫柔巴結：「你如今已然筑基圓滿，若渡過九重雷劫，從此仙途無量。可別忘了我當初贈丹相助的情分。」
7. 兩大核心玩法模組（嚴格分層）
7.1 前期：境界突破養成（無戰鬥、無劫難）
- 核心行為：打坐積累修為、閉關突破
- 核心風險：隨機運氣判定、突破失敗掉級、丹藥白白消耗
- 核心樂趣：隨機師姐劇情、爽文人情拉扯、穩步養成進階
7.2 後期：九重雷劫渡劫操作（遊戲核心高難玩法）
- 解鎖條件：筑基9層圓滿
- 玩法機制：分段式九重雷劫、閃躲判定、階段性QTE、高容錯高刺激操作
- 成敗結果：
成功：突破大境界、解鎖全新養成內容
失敗：境界跌落、需重新積累修為，體現修仙真實風險
8. 規則最終統整（徹底解決所有邏輯矛盾）
1. 練氣全程：零劫難、零雷罰，完全符合原著邏輯
2. 筑基初期突破：吃丹突破，僅有隨機運氣成敗機制，無丹毒、無渡劫
3. 渡劫玩法只存在：筑基九層圓滿之後
4. 師姐贈丹只給「突破機會」，不給「通關保障」，邏輯自洽、爽點充足
5. 無多餘系統：無煉丹、無材料收集、無禁地副本，玩法精簡聚焦
9. 遊戲亮點總結
1. 邏輯嚴謹：徹底避開「練氣渡劫」的不合理bug，仙階規則清晰統一
2. 爽文氛圍到位：隨機心機師姐NPC，人情投資、雙標反差，玩家共鳴感極強
3. 玩法層次分明：前期休閒養成看劇情，後期操作渡劫考技術
4. 輕量精簡：無雜項系統，開發成本低、玩家上手門檻低
5. 可拓展性強：後續可持續新增不同性格隨機師姐、新增劫難階段、新增境界內容
